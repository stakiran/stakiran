## Objective
新しいポジション「ナレッジ・アーキテクト」を希望します。これはエンジニアとマネージャーを含む全社員向けにソフトスキルを開発、発信、啓蒙するとともに、組織やプロジェクト全体のナレッジ開発および共有の仕組みをリードする役割です。
